Website design

website uses modern coventions when making website such as using grid area to design the layout of the website which is a modern standard when creating a website.The website also uses multiple stylesheets for each page to suit both viewing on a computer or laptop and to suit viewing on a mobile device as the style changes to a more mobile friendly version with a hamburger menu button.

Proper tags have been used to layout the website using tags such as main, nav, aside, header, footer, etc. These make the website more organised as well as easier for desktop narrators to understand as using tags like main and nav can make it easier for them to interpret the website.

mobile site

The website mobile versions have a hamburger icon. This icon is commonly used on mobile sites as a way to open a side menu with the websites navigation as to not clutter the screen and making it a more viewable experience. This is done on this website through target selectors that detect when a link is clicked and can change the style of the website through CSS.

Contact us page and forms

The contact us website uses a form to create a submission box where you can enter details and a question. This is supposed to simulate an actual contact us page. However, the form does not have the functionality of a contact form as that would require javascript.

keyframe animation

A keyframe animation is used on this website to demonstrate how to move objects around the page using keyframes to animate its movement and change its properties during the animation.
